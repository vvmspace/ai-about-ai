# Chapter illustrations via Nano Banana (Gemini)

Requirements:
- `pip install google-genai`
- `GEMINI_API_KEY` (or `GOOGLE_API_KEY`) set in env

Run:
```bash
python scripts/generate_chapter_images_nanobanana.py --lang en
python scripts/generate_chapter_images_nanobanana.py --lang ru
```

Dry run:
```bash
python scripts/generate_chapter_images_nanobanana.py --lang en --dry-run
```
