# Goal

Написать книгу для подготовки к интервью по заданной теме.
Книга должна охватывать весь спектр потеницальных вопросов.

## Style Guide (mandatory voice)

### Goal
Produce English that feels like a modern upper-class British “educated Southern” register with ex-military composure: restrained, precise, quietly authoritative.

### Core vibe
- Quiet confidence; minimal surface emotion.
- Polite dominance: civil tone, firm terms.
- Understatement over hype.
- Dry, minimal humour.

### Delivery
- Medium-slow, deliberate pace.
- Crisp diction.
- Statements should “land” clearly.

### Sentence shape
- Short, decisive sentences.
- One idea per sentence where possible.
- Controlled connectors: “However.” “In any case.” “That said.”
- Prefer triads: “Time. Cost. Risk.”

### Lexicon guidance
- Understatement: rather, quite, somewhat, in practice, in principle.
- Precision: strictly speaking, to be precise, the core issue is.
- Control frames: “Let’s be clear—”, “Here’s what’s going to happen—”.
- Polite refusal: “I’m afraid not.” / “That won’t work.”

### Grammar signatures
- Present perfect for current status.
- Present perfect continuous for ongoing effort.
- Past simple for finished, time-marked events.
- Modals for controlled authority: would / could / might / need to.

### Avoid
- Emotional pleading and dramatics.
- Filler words.
- Excessive enthusiasm.

---

Book structure:
- Оглавление: список глав, по одному предложению после каждой главы. Начинай писать с него, чтобы продумать модель книги.
- Книга должна быть захватывающая в целом.
- Книга должна быть легко усваиваемая для любого уровня читателя.
- Примеры приведенные в книге должны быть простыми, забавными, чтобы человек быстро понимал зачем ему это нужно.


Chapter structure:
- Ошарашить
- Озадачить (удержать)
- Пример ответа на интервью с цитатой на английском
- Пара красивых фраз для запоминания
- Дофаминоэндорфиновая петля, чтобы было желание читать дальше
