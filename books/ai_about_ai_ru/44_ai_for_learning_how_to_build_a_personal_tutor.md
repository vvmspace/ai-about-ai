# AI для обучения: как собрать персонального тьютора

ИИ для обучения работает лучше всего как активный coach, а не «ответник».

## Архитектура тьютора

explain -> quiz -> diagnose gaps -> practice -> spaced revision.

## Базовый prompt-loop

«Объясни тему, проверь меня 5 вопросами, найди 2 слабых зоны, дай 20-минутный план тренировки».

## Ошибки

- пассивное чтение,
- отсутствие retrieval practice,
- нет журнала ошибок,
- нет графика повторений.

## Главная мысль

Учёба с ИИ = интерактив + диагностика + повторяемый ритм.
