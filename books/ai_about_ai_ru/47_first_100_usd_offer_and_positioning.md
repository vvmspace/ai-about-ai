# Первые $100 с AI — Глава 1: оффер и позиционирование

Сначала вам не нужен сайт.
Вам нужен один оплаченный результат.

Всё остальное вторично.

Когда люди не достигают первой цели по выручке,
это редко проблема инструментов.
Обычно они упаковали расплывчатое обещание,
а потом удивились,
почему покупатели медлят.

Английская версия этой главы:
[First $100 with AI — Chapter 1: Offer and Positioning](../ai_about_ai/47_first_100_usd_offer_and_positioning.md).

## Что покупатели реально покупают на старте

Никто не платит за «помощь с AI».
Платят за быстрое снятие конкретной боли.

Если оффер звучит широко,
риск ощущается высоким.
Если оффер звучит точно,
трение принятия решения падает.

В этом и вся игра.

## Ловушка новичка: «я умею всё»

Широкое меню кажется гибким.
Для клиента
оно выглядит неопределённым.

Какие сигналы посылает широкий оффер:

- неясная экспертиза,
- неясный процесс,
- неясный стандарт результата,
- неясная зона ответственности.

Неясность стоит дорого,
поэтому покупатель откладывает.

## Формулировка оффера, которая должна помещаться на один экран

Используйте такую структуру:

«Я даю [результат]
для [ниши]
за [срок]
за [фиксированную цену]».

Пример:
«Я делаю 5 рекламных креативов
для локальных фитнес-студий
за 48 часов
за $120».

Без жаргона.
Без раздутого бренд-языка.

## Выбирайте нишу по повторяемости, а не по личной идентичности

Ваша первая ниша должна иметь:

- много похожих бизнесов,
- регулярные потребности в контенте или продажах,
- заметные проблемы качества,
- быстрые циклы согласования.

Хорошие стартовые примеры:

- локальные клиники,
- салоны,
- брокеры недвижимости,
- независимые коучи,
- небольшие ecommerce-магазины.

Вы не вступаете в брак с нишей.
Вы покупаете быстрое обучение.

## Выберите одну болезненную задачу и перестаньте добавлять функции

Ранний scope должен быть узким.
Выберите одну задачу, например:

- пакет рекламных креативов,
- переписывание лендинга,
- набор сообщений для outreach,
- пакет коротких видео,
- обновление описаний товаров.

Одна задача.
Один результат.
Один дедлайн.

Чем больше вы добавляете,
тем менее убедительно выглядите.

## Проверка прочности оффера

Перед outreach проверьте пять пунктов:

1. Покупатель понимает это за 10 секунд?
2. Результат можно оценить визуально или по тексту?
3. Срок доставки явно указан?
4. Цена фиксирована для пилота?
5. Можете быстро показать мини-демо?

Если на два пункта ответ «нет»,
оффер не готов.

## Тон позиционирования: спокойная определённость

Избегайте извиняющейся позиции.
Избегайте агрессивной позиции.

Используйте профессиональную прямоту:

«Вот scope,
срок,
цена
и политика правок.
Если вам подходит,
начинаем сегодня».

Вежливо.
Ясно.
Сложно неправильно понять.

## Соберите мини-портфолио без перепроизводства

Вам не нужны тридцать примеров.
Вам нужны три релевантных доказательства:

- пример «до/после»,
- пример вариаций A/B,
- пример в вашей нише.

Соберите это в один PDF или страницу Notion.

Покупатели оценивают доказательства,
а не автобиографию.

## Workflow доказательств за один вечер

1. Выберите 2–3 реальных бизнеса в вашей нише.
2. Сделайте по одному демо-артефакту для каждого.
3. Пометьте всё как концептуальную работу.
4. Упакуйте материалы в один аккуратный файл.

Готово.
Теперь можно продавать.

## Границы защищают маржу и психику

Сразу озвучьте исключения:

- без ведения рекламных кабинетов,
- без гарантий ROI,
- без безлимитных правок,
- без задач вне scope без нового расчёта.

Для ясности:
нечёткие границы создают неоплачиваемый труд.

## Политика правок для первого оплаченного пилота

Используйте простой стандарт:

- один первоначальный драфт,
- два раунда правок,
- дополнительная работа оплачивается отдельно.

Это не жёсткость.
Это операционная гигиена.

## Почему фиксированная цена выигрывает на старте

Почасовая оплата требует доверия,
которого у вас пока нет.

Фиксированная цена пилота упрощает решение:

- понятный нижний риск,
- понятный deliverable,
- понятные сроки.

Для первых сделок
ясность важнее оптимизации.

## Рекомендуемая входная цена для первой выручки

Для большинства стартовых офферов:

- пилот за $50–$150,
- выполнение за 24–48 часов,
- намеренно узкий scope.

Первая цель — валидация,
а не максимальное извлечение.

## Сообщение, которое можно отправить сегодня

«Здравствуйте, [Имя],
я заметил(а) [конкретное наблюдение].
Я делаю [результат] для [ниши]
за [срок] за [$X].
Могу отправить один концепт сегодня.
Будет полезно?»

Короткие сообщения конвертируют лучше,
чем эссе.

Конкретный оффер,
ясная ниша,
фиксированный scope,
видимые доказательства,
чёткие границы.
