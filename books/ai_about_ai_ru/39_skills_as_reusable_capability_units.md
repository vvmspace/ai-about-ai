# Skills как переиспользуемые capability-юниты

Одинокие промпты не масштабируются. Skills — масштабируются.

## Сильный skill содержит

- scope,
- trigger,
- входы,
- последовательность tool-calls,
- выходной контракт,
- fallback.

## Жизненный цикл

pain -> рабочий workflow -> упаковка -> валидация -> версионирование.

## Главная мысль

Skills — мост между «умной моделью» и предсказуемым исполнением.
