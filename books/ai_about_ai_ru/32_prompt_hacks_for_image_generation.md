# Фишки промптов для генерации изображений

Хороший image-промпт — это арт-дирекшн, а не поэзия.

## Рабочая структура

subject -> scene -> composition -> lighting -> style -> technical -> negatives.

## Что даёт резкий прирост

- конкретная композиция (rule of thirds, symmetry, macro),
- явный свет (rim light, golden hour, volumetric haze),
- короткий блок tactical negatives (watermark, extra fingers, blur).

## Итерационный цикл

1) идея (варианты), 2) фиксация кадра/света, 3) полировка деталей.

## Главная мысль

Точность + итерации + коммерческая применимость побеждают «магические промпты».
