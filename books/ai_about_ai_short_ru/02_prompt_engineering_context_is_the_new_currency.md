# Контекст — новая валюта

Если из главы нужно унести только одну мысль, пусть будет эта:

## Почему команды ошибаются в диагнозе

Когда ответ кажется поверхностным, обычно винят интеллект.

## Архитектура контекста: четыре слоя

Сильные команды проектируют **контекст** слоями, а не свалкой документов.

## Стратегия контекста — это не «запихнуть побольше»

Запросы вроде «стратегия контекста для ИИ» и «как правильно давать контекст ChatGPT» часто ведут к ошибочной идее: чем больше текста, тем лучше.

## Паттерн context packet

Для повторяющихся задач сделайте reusable packet:

- цель в одной строке;
- обязательные ограничения;
