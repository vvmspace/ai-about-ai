# 37. Unique Paths: одна сетка, две строгие модели решения

Эта глава полезна тем, что показывает зрелость мышления: одну и ту же задачу вы можете решить и через комбинаторику, и через динамическое программирование.

Модель первая — dynamic programming (динамическое программирование):
- `dp[r][c]` — число путей до клетки `(r, c)`;
- из каждой клетки можно прийти только сверху или слева;
- переход: `dp[r][c] = dp[r - 1][c] + dp[r][c - 1]`.

Модель вторая — combinatorics (комбинаторика):
- всего шагов `m - 1 + n - 1`;
- из них нужно выбрать, где будут вертикальные шаги;
- ответ: биномиальный коэффициент `C(m + n - 2, m - 1)`.

Сложность:
- динамическое программирование: время **O(m × n)**, память **O(n)** при оптимизации по строке;
- комбинаторика: время **O(min(m, n))**, память **O(1)**.

Термины:
- **grid state** — состояние клетки в сетке;
- **binomial coefficient** — число сочетаний без порядка;
- **equivalent models** — разные математические модели одного результата.

Пример на Rust (оптимизированное динамическое программирование):

```rust
pub fn unique_paths(m: i32, n: i32) -> i32 {
    let rows = m as usize;
    let cols = n as usize;
    let mut dp = vec![1; cols];

    for _ in 1..rows {
        for c in 1..cols {
            dp[c] += dp[c - 1];
        }
    }

    dp[cols - 1]
}
```

Интервью-ответ:

> “I can solve it with dynamic programming on the grid, and I can also derive the same result combinatorially as a binomial coefficient. I usually present dynamic programming first for readability.”  
> [«Я могу решить задачу динамическим программированием по сетке, а также вывести тот же результат комбинаторно как биномиальный коэффициент. Обычно сначала показываю динамическое программирование как более читаемую версию».]

Фразы для закрепления:
- “Two models. One truth.”  
  [«Две модели. Один результат».]
- “Choose clarity, then elegance.”  
  [«Сначала ясность, потом элегантность».]

Где это в работе: подсчёт маршрутов в логистике, оценка числа сценариев в пошаговых процессах, расчёт вариантов прохождения пайплайна.
